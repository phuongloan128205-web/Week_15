# 🏠 House Price Prediction Project

## 📌 Description

This project predicts house prices based on input features such as area and number of bedrooms using Machine Learning.

## 🚀 Features

* Input house details
* Predict price instantly
* Simple web interface using Streamlit

## 🧠 Technologies

* Python
* Scikit-learn
* Streamlit

## ⚙️ How to Run

pip install -r requirements.txt
python train.py
streamlit run app.py

## 📊 Example

Input: area = 70, bedrooms = 3
Output: 1200 (million VND)
